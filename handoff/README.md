# NEO PM ダッシュボード — Claude Code 引き継ぎパッケージ

> このフォルダは、HTML プロトタイプを **neo-pm Next.js コードベース** に実装するための仕様一式です。Claude Code に渡してください。

## このパッケージで決まっていること

- **方向性**: `C v2 — Gamified Glass`（淡青 × フロストガラス）
- **既定設定**: `mood=cool / frost=crisp / fun=festive`
- **対象画面**: 9 枚（ランキング / ダッシュボード / 実行計画 / WBS / 収支 / 診断 / 基金申請 / AI伴走 / テーマ出題）
- **常駐 UI**: 右下 浮遊 AI 伴走者（全画面に存在）
- **AI 動作**: 提案カード型（提案 → ✓ 承認 → 各画面に反映）

## このパッケージで決まっていないこと（実装側で決める）

- フレームワーク（Next.js / Tailwind / shadcn 推奨だが指定なし）
- 状態管理（React Context で十分）
- API 実装（仕様は `05-data-model.md` を参照、実装は別途）
- 認証・権限（既存 neo-pm 流儀に従う）

## プロトタイプ参照

`prototype/` フォルダに、このパッケージのもとになった HTML / JSX 一式があります。

- `NEO PM Redesign v2.html` — 全画面が動くデモ
- `variant-c-v2.jsx` — 主要 5 画面（ランキング / ダッシュ / 実行計画 / AI / テーマ）+ 浮遊 AI
- `variant-c-v2-extra.jsx` — 追加 4 画面（WBS / 収支 / 診断 / 基金申請）
- `styles.css` — 共有トークン（最重要）
- `data.jsx` — モックデータ

ローカルで `index.html` を開けば、Tweaks も AI 提案カードも動きます。

## 注意点

- **インライン CSS が多い**: プロトタイプは "1ファイルで完結" 優先で書いたため、本実装では `<GlassCard>` などにまとめてください
- **絵文字をアイコンに**: 🚀 🎯 などは Lucide / Heroicons に置き換え推奨（モードによっては絵文字も可）
- **アクセシビリティ**: フォーカスリング、aria-label、コントラスト比は実装側で担保
- **モバイル**: プロトは 1400×920 固定。レスポンシブ対応は実装フェーズで設計
